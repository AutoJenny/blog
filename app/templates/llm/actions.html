How to Create and Use LLM Actions

  1. Add a New Action: Click New Action and fill in the basic settings (field name, model, etc.).
  2. Edit Prompt Parts: Click Prompt Parts to build your prompt from multiple parts. (See below for detailed instructions.)
  3. Test Your Action: Click Run/Test to try it out with sample input.
  4. Use in Posts: Select your action in the post editor and click Generate to use it for real content.
Tip: Use Prompt Parts to break your prompt into logical steps (system, user, assistant, etc.) for better LLM results. You can use variables like {{input}} or {{idea_seed}} in your prompt parts.

Prompt Parts Editor

  • Prompt Parts are the building blocks of your LLM prompt. Each part can be a System (instructions), User (input), or Assistant (expected response).
  • Add parts in the order you want them sent to the AI. Use System for instructions, User for the main request, and Assistant for expected responses (optional).
  • You can use variables like {{input}} or {{idea_seed}} to insert data from your post.
  • Drag parts to reorder, or use the up/down arrows.
  • The Preview panel shows how your full prompt will look to the AI.
